● About This Project

Built with ML rigour,
designed for clarity

🎓

Capstone Project

Opportunity Scout AI was built as a data science capstone, demonstrating a full production ML pipeline from raw data collection through to a deployed, explainable scoring engine.

🤖

The Model

A LightGBM model trained on 25,000+ historical ventures.

🔎

Honest Explainability

SHAP attributes every score point to a specific feature. Founding year does not penalise your score, the model evaluates market conditions which are timeless. Training data spans 2005–2020.

📡

Signal Sources

Google Trends for market momentum, World Bank for GDP and inflation, Reddit for community density, GDELT for news sentiment with historical CSV benchmarks as fallback when live APIs are unavailable.
